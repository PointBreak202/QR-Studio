# QR Studio

A playful, modern QR code generator built using Flask, HTML, CSS and JavaScript.

## Features

- Generate QR codes instantly
- Live QR preview
- Download PNG
- Copy URL
- Responsive UI
- Cute pastel design

## Tech Stack

- Flask
- Python
- HTML
- CSS
- JavaScript
- qrcode
- Pillow
